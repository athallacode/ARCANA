   $ cd Test
   $ pip install -r requirements.txt

2️⃣  RUN THE APP:
   $ python digital_whiteboard.py

3️⃣  START TESTING:
   • Tulis huruf "A" atau "BA" di papan tulis

   • Lihat hasil pengenalan AI di panel kanan

Error: OCR not recognizing text
→ Write more clearly, avoid scribbles
→ Use thicker strokes
→ Avoid overlapping characters
